jQuery

Posted on 2013-03-26 | Post modified 2017-06-11 | In Libraries

jQuery makes rich UI development fast, easy and cross-browser compatible by providing a fast and concise JavaScript library that simplifies:

  • DOM traversal and manipulation
  • Event handling
  • UI effects and -manipulation
  • AJAX

The functionality in jQuery could be broken down into eight major categories:

  • Core Functionality. jQuery core functions and commonly used utilities.
  • Selection and traversal. Provides functions for finding and navigating content in a document i.e. DOM traversal.
  • Manipulation & CSS. Functions for changing content and working with CSS data.
  • Events. Provides functions and helpers for working with DOM events
  • Effects. Functions for creating basic animations and effects like hide/show and moving elements on the screen.
  • Ajax. Provides AJAX functions for loading content and JSON data handling.
  • User Interface. Provides an official plug-in with commonly used interface widgets.
  • Extensibility. Enables enhancement of base functionality via plug-ins

Selectors and Filters

Selectors and filters represent the query part of jQuery. They are used to select and filter out content that then can be further manipulated using other functions.

Selectors

jQuery Selectors return a list of objects matching the selection criteria provided. The selectors return collections of jQuery objects, not DOM elements. Selectors and filters use familiar CSS syntax.

SELECTOR PURPOSE
tagname Finds all elements with the provided tagname
#identifer Finds element with ID
.classname Finds all elements using the specified class
tag.classname Gets elements with the provided tag that use the specified class
* Finds all the elements in a page
sel1,sel2… Finds all the specified selectors
class1.class2 Finds all selectors with both class1 and class2 applied
parent > child Finds child elements that are direct children of elements of type parent
ance desc Finds all descendant elements that are contained within elements of type ancestor
prev+next Finds all next elements that are next to a prev element
prev ~ sibl Finds all sibling elements that come after prev and match the siblings selector

Filters

Filters operate on selectors to further refine the result array that the selector returned.

Six categories of filters:

  • Basic. Basic filtering like getting first, last, even or odd numbered items.
  • Content. Filters a set of elements based on their content, like wether an element contains a particular string.
  • Visibility. Filters a set of elements using the visibility setting of each element as a test.
  • Attribute. Filters based on an element’s attributes.
  • Child. Selects elements based upon their relationship with their parent element.
  • Form. Provides specialized filters on form elements.

Basic Filters

Refine the results of a selector by including elements matching some condition like position or index:

FILTER PURPOSE
:first Selects only the first element of the selector’s returned set
:last Selects only the last element of the selector’s returned set
:even Selects even-numbered elements in the selector’s returned set
:odd Selects odd-numbered elements in the selector’s returned set
:eq(n) Filters out the elements positioned at the given index
:gt(n) Includes elements that appear after the given index
:lt(n) Includes elements that appear before the given index
:header Selects all header elements (H1..H6)
:animated Selects all elements currently being animated
:not(selector) Includes elements that does not match the given selector

Attribute Filters

FILTER PURPOSE
[attribute] Includes elements that has the specified attribute
[attribute=value] Includes elements if they have the specified attribute and value
[attribute!=value] Include elements if they have the specified attribute and it doesn’t have the specified value
[attribute\^=value] Include elements if they have the specified attribute and it starts with the specified value
[attribute$=value] Include elements if they have the specified attribute and it ends with the specified value
[attribute*=value] Include elements if they have the specified attribute and it contains the specified value
[attrFlt1] [attrFltN] Includes elements that mach all of the specified attribute filters

Content and visibility filters

FILTER PURPOSE
:contains(text) Filters the selection to only include elements containing given text
:empty Filters the selection to only include empty elements i.e. has no children
:has(selector) Matches elements that contain at least one element that has the given selector
:parent Matches all the elements that are parents (contains at least one other element, including text)
:visible Filters the selection to only include visible elements
:hidden Filters the selection to only include hidden elements

Child filters

FILTER PURPOSE
:nth-child(index) Matches elements at index
:nth-child(even), nth-child(odd) Matches even or odd elements
:nth-child(equation) Match an equation of the form Xn+M (like 2n or 3n+1)
:first-child Matches element that is the first child of their parent
:last-child Matches element that is the last child of their parent
:only-child Matches elements who are the only child of their parent

Form Selectors

SELECTOR PURPOSE
:input Finds all input, select, textarea and button elements
:text Finds all text elements
:password Finds all password elements
:radio Finds all radio button elements
:checkbox Finds all checkbox elements
:submit Finds all submit elements
:reset Finds all reset elements
:image Finds all image elements
:button Finds all button elements
:file Finds all file upload elements
:enabled Matches all elements that are enabled
:disabled Matches all elements that are disabled
:checked Matches all form elements that are checked (radiobuttons and checkboxes)
:selected Matches all form elements that are selected

Traversing Document Information

Function/Property Purpose
size(), length The number of elements in a jQuery result set
get() Returns an array of matched DOM elements. For when you need to work on DOM elements instead of jQuery functions
get(index) Access a single matched DOM element at the specified index in a result set
find(expression) Searches for descendant elements that match the specified expression
each(fn) Execute a function within the context of every matched element

Statement chaining

One of jQuery’s most powerful features is its ability to chain multiple functions together to perform several operations in one line of code.

Manipulating Content

Once you’ve used selectors and filters to retrieve web page content, you usually want to manipulate it.

Sometimes you want to create new content dynamically into the page. jQuery has functions for creating, copying, deleting and moving content around as well as wrapping page content in other content.

Creating new content

To create new HTML content, you simply pass a string containing new HTML to the $() function:

In addition you can use the html() and text() functions to get and set content on elements.

Function Purpose
html() Returns the HTML content of the first matched element
html(newcontent) Sets the HTML content of every matched element
text() Returns the text content of the first matched element
text(newtext) Sets the text content for all matched elements

Manipulating Attributes

To change or inspect attribute values on elements use the attr() function.

Function Purpose
attr(name) Accesses property on the first matched element. If no such attribute exists, undefined is returned
attr(properties) Sets attribute values on all matched elements using object notation syntax. Use for setting a large number of properties at once
attr(key, value) Sets a single property to a value on all matched elements
attr(key, fn) Sets a single property to a computed value on all matched elements
removeAttr(name) Removes the named attribute from all matched elements

Inserting Content

There are several functions for inserting content into the document both before and after existing page elements.

Function Purpose
append(content) Appends content inside every matched element
appendTo(selector) Appends all of the matched elements to another, specified, set of elements
prepend(content) Prepends content to the inside of every matched element
prependTo(selector) Prepends all the matched elements to another, specified, set of elements
after(content) Inserts content after each of the matched elements
before(content) Insert content before each of the matched elements
insertAfter(selector) Inserts all of the matched elements after another, specified, set of elements
insertBefore(selector) Inserts all the matched elements before another, specified, set of elements

Wrapping, Replacing and Removing Content

Function Purpose
wrap(html) Wraps each matched element with the specified HTML content
wrap(element) Wraps each matched element with the specified element
wrapAll(html) Wraps all the elements in the matched set with the specified HTML content
wrapAll(element) Wraps all the elements in the matched set into a single wrapper element
wrapInner(html) Wraps the inner child contents of each matched element (incl. text nodes) with a HTML structure
wrapInner(element) Wraps the inner child contents of each matched element (incl. text nodes) with a DOM structure
replaceWith(content) Replaces all matched elements with the specified HTML or DOM elements
replaceAll(selector) Replaces the elements matched by the specified selector with the matched elements
empty() Removes all child nodes from the set of matched elements
remove() Removes all matched elements from the DOM
clone() Clone the matched DOM elements and select the clones
clone(bool) Clones the matched DOM elements, and all their event handlers, and select the clones

Manipulating CSS

jQuery’s CSS functions provide easy cross-browser access for setting properties, working with positioning and sizing values.

The css() function allows you to retrieve and set CSS styles for a set of matched elements.

Function Purpose
css(name) Returns the value for the named CSS property for the first matched element
css(properties) Sets the CSS properties of every matched element using object notation syntax
css(property,value) Sets a single style property to a given value on all matched elements. Nr values are converted to pixels but do not work on all attributes

jQuery also provides a set of functions for working with CSS classes on page elements. Classes can be added, removed, toggled and detected.

Function Purpose
addClass(class) Adds specified class(es) to each matched element
hasClass(class) Returns true if the specified class is present on at least one of the set of matched elements
removeClass(class) Remove all the specified class(es) from the set of matched elements
toggleClass(class) Adds the specified class if not present, removes it otherwise
toggleClass(class,switch) Adds the specified class if the switch is true, removes it otherwise

CSS Positioning

Function Purpose
offset() Gets the current offset of the first matched element, in pixels, relative to the document
offsetParent() Returns a jQuery collection with the positioned parent of the first matched element
position() Gets the top and left position of an element relative to its offset parent
scrollTop() Gets the scroll top offset of the first matched element
scrollTop(val) Sets the scroll top offset to the given value on all matched elements
scrollLeft() Gets the scroll left offset of the first matched element
scrollLeft(val) Sets the scroll left offset to the given value on all matched elements

CSS Sizing

Function Purpose
height() Gets the current height in pixels of the first matched element
height(val) Sets the CSS height attribute for every matched element
width() Gets the current width in pixels of the first matched element
width(val) Sets the CSS width attribute for every matched element
innerHeight() Gets the inner height (excl. border but incl. padding) of the first matched element
innerWidth() Gets the inner width (excl. border but incl. padding) of the first matched element
outerHeight(bool) Gets the outer height (incl. border and padding by default) for the first matched element. If bool argument is true, the margin value is also included
outerWidth(bool) Gets the outer width (incl. border and padding by default) for the first matched element. If bool argument is true, the margin value is also included

Events

jQuery provides a mechanism for working with events that is simpler than the DOM event model. It also abstracts away the differences between browser implementations. jQuery makes it easy to assign event handlers to groups of elements by using selectors and filters.

Event Categories

Binding/Unbinding

Allows events to be wired up and torn down in a cross-browser fashion. Events are connected to and disconnected from elements using the bind() and unbind() functions.

Bind Params
Param Purpose
event Defines the event that you want to be bound to for each element in the selector’s result set. Possible values are blur, focus, resize, scroll, unload, before, unload, click, dblklick, mousedown, mouseup, mousemove, mouseover, mouseout, mouseenter, mouseleave, chenge, select, submit, keydown, keypress, keyup, error
data Optional. Defines a piece of data that will be passed to the handler function when the event occurs and the handler function is called
handler Specifies the function that will handle the event
Unbind Params
Param Purpose
event Defines the event that you want to be disconnected for each element in the selector’s result set
handler Specifies the handler function that was defined to handle the event

Unified Event Object

Provides an event object that exposes the most common properties in a cross-browser fashion. Most common properties on the event object:

Function Purpose
type Type of the event (‘click’)
target Element that issued the event
data Data passed to bind function
pageX, pageY Mouse coordinates relative to document
result Value returned by the last handler function
timestamp Time when the event occurred
preventDefault() Prevent the browser from executing the default action
isDefaultPrevented() Returns whether preventDefault() was ever called on this object
stopPropagation() Stops the bubbling of an event to parent elements
isPropagationStopped() Returns whether stopPropagation() was ever called on this object
